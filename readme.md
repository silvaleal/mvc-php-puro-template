# MVC PHP Template

Um template MVC (Model-View-Controller) simples e eficiente para projetos PHP.

## 🚀 Instalação

```bash
git clone https://github.com/silvaleal/mvc-php-template.git
cd mvc-php-template
composer install
cp .env.example .env
php -S localhost:8000 -t public
```

## 🏗️ Estrutura do Projeto

```
mvc-php-template/
├── app/
│   ├── Controllers/
│   └── Helpers/
│   └── Middlewares/
│   └── Models/
│   └── Views/
├── database/
├── public/
├── routes/
```
